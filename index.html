HTML

  • The information, or metadata, in the head tag is not visible tot he user
  • The body element represents the visible content shown to the user.
  • The most important headers on each page get the h1 tag and the subsequent headings recieve h2-h6 tags
  • Paragraphs and other text usually goes between the p tags

Git

  • git status: checks what branch you are currently in.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git remote -v: should display the URL of your GitHub repository so you can confirm your local repository is connected.
  • git pull origin main: synchronizes your local and remote repositories after you make changes.
  • git branch: lists all the branches in your local directory.
  • git push origin main: pushes or uploads you changes to your remote repository.
  • cd directory name: changes the directory you are in.
  • cd: moves you up or back in the directory hierarchy.
  • ls: lists all the files and directories in the current directory.
  • pwd: displays the current direcctory's full path.
  • mkdir new folder name: creates a new directory
