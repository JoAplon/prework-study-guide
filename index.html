HTML

  • The head element contains information about the webpage.
  • The information, or metadata, in the head tag is not visible tot he user
  • The body element represents the visible content shown to the user.
  • The most important headers on each page get the h1 tag and the subsequent headings recieve h2-h6 tags
  • Paragraphs and other text usually goes between the p tags

CSS

  • There are three ways to style a webpage with css: inline, internal, and external.
  • Inline styles apply directly to a specific html tag. It uses a style attribute with a css rule. They are useful for quick and permanent changes, but less flexible than external and internal since inline must be edited individually.
  • Internal style sheet contains css rules in the head section of the html file. It will apply to only one page and uses classes and ids.
  • External is the most common way to styles webpages since it allows developers to keep the css rules in an separate file. This method uses a link element in the head section.
  • A margin property indicates how much space we want around an object.
  • A padding property adds space around the content inside and element. This makes the entire element bigger.
  • You can combine two selectors into a single rule with a comma so the declarations affect both selectors. This can look like header, footer then the declaration.
  • A period before a class name designates it as a class selector. Any element assigned to the class will will gain these css property assignments.

Git

  • git status: checks what branch you are currently in.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git remote -v: should display the URL of your GitHub repository so you can confirm your local repository is connected.
  • git pull origin main: synchronizes your local and remote repositories after you make changes.
  • git branch: lists all the branches in your local directory.
  • git push origin main: pushes or uploads you changes to your remote repository.
  • cd directory name: changes the directory you are in.
  • cd: moves you up or back in the directory hierarchy.
  • ls: lists all the files and directories in the current directory.
  • pwd: displays the current direcctory's full path.
  • mkdir new folder name: creates a new directory
